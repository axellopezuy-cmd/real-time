1 Comando para empezar

Un comando. Eso es todo.

No hay webpack. No hay bundler. No hay node_modules. Solo tu código y el resultado.

1

Instalá

curl -fsSL https://realtime.dev/install.sh | sh

Un binario estático. Sin dependencias. Sin runtime.

2

Ejecutá

realtime ./mi-proyecto

Apuntá a cualquier carpeta con archivos HTML y CSS.

3

Desarrollá

Abrí tu editor, escribí código. El browser se actualiza en tiempo real mientras tipeás. Cada tecla, cada cambio, al instante.

Todo lo que necesitás

Construido para devs que valoran velocidad y simplicidad

🦀

Rust nativo

Server compilado en Rust con Axum. Binario estático, sin runtime, sin garbage collector. Arranca en milisegundos.

⚡

WASM Parser

Parsers de HTML y CSS compilados a WebAssembly (33KB gzip). Corren en el browser a velocidad near-native para overlays y análisis.

🔄

DOM Morphing

No hacemos innerHTML. Diffing inteligente del DOM que solo toca los nodos que cambiaron. Cero parpadeo, cero flicker.

🎨

CSS real

Grid, Flexbox, variables, animaciones, media queries — todo funciona. Es un browser real, no una simulación.

🔍

Inspector integrado

Outlines y labels como Chrome DevTools. Activá overlays para ver la estructura de tu layout sin salir de Real Time.

📡

WebSocket nativo

Conexión persistente con reconexión automática. Solo envía los archivos que cambiaron (delta updates).
